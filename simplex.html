Simplex method solver

This webpage essentially just solves a linear programming problem specified in matrix form using the simplex method and shows full working. The problem must be in canonical matrix form, that is: \[ \begin{aligned} &\mathrm{Maximize\hspace{0.1cm}z} &&= \mathbf{c}^T \mathbf{x} \\ &\mathrm{Subject\hspace{0.1cm}to:}\\ &A\mathbf{x} &&= \mathbf{b} \\ &\mathbf{x} &&\geq 0 \end{aligned} \] The various matrices can be entered in using MATLAB notation. In the case of A, semicolons are used to separate rows and commas or spaces to separate individual row elements. \(\mathbf{b}\), \(\mathbf{c}\), \(\mathbf{x}\) and \(x_{\mathbf{B}}\) can have individual elements separated by commas, semicolons or spaces.
